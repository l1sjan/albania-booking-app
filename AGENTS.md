# Project: Albania Service & Booking Hub
An app for locals and tourists in Albania to book services (car rentals, barbers, dentists) and subscribe businesses.

## Tech Stack
- **Frontend:** Flutter (Dart) - supports iOS/Android, localizable (Albanian/English)
- **Backend/Database:** Supabase (PostgreSQL)

## Rules of Engagement
1. **Always plan first:** Use the `create-plan` skill to write down the implementation steps before editing files.
2. **Keep code modular:** Separate UI views, state management, and API clients.
3. **Bilingual by default:** Keep all user-facing strings separated into localization files (Albanian/English).
4. **No local database mockups:** Hook directly into Supabase client SDK structures from day one.
